\section{Automated Assets and Figure Sources}

The manuscript tooling automates only the parts of the draft that currently
have stable, canonical inputs. At the moment that means a clean-tree Lane~A
holdout figure and table, a headline anti-recomputation figure and table, a
sanitized repo-provenance table, an appendix-only deployment evidence map, and
an imported contact sheet from the streaming evaluation when the expected
live-demo images are available. The sync step avoids derived paper-side
summaries, avoids workstation-local absolute paths, and excludes dirty-tree
supplementary rows from the default automated snapshot.

Several paper elements are still intentionally manual. The novelty-ranked dense
baseline table is currently carried from the authoritative phase note because a
stable checked-in artifact path is not yet wired into the paper tooling. The
supporting systems section also still uses hand-written summary prose rather
than generated tables, because those claims are still being sorted into
main-text versus appendix roles. This is deliberate: the draft automates what
is stable and leaves the moving pieces visibly manual.
